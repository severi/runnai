RunnAI

An AI running coach that actually remembers you.

Open-source coaching agent that connects to Strava, learns your patterns, builds adaptive training plans, and gets smarter every session. All your data stays on your machine.

What it does

RunnAI is a coaching agent that runs in your terminal. It connects to your Strava, builds a picture of who you are as a runner, and uses that context in every conversation. Coaching advice is grounded in running science, not generic LLM output.

  • Persistent memory Remembers your injury history, training preferences, and how you respond to different workloads. The coach's picture of you evolves over sessions - not a blank slate every time.
  • Strava sync and write-back Syncs your activities, classifies every run, and writes coaching notes back to your Strava descriptions - the kind of feedback a human coach would give.
  • Deep run analysis Every run is broken down from per-second data: HR zones, cardiac drift, grade-adjusted pace, fatigue index, workout phases, and interval detection.
  • Adaptive training plans Plans that adjust weekly based on what you actually ran, with export to intervals.icu as structured workouts with HR/pace targets and color-coded tags.
  • Race predictions Estimates that evolve as your fitness changes, tracked over time so you can see the trend heading into a goal race.

Intervals.icu integration

Training plans export directly to intervals.icu as structured workout events. Each workout gets HR/pace targets, color-coded tags, and repeat blocks that intervals.icu renders as visual workout steps. Re-exporting the same plan safely updates existing events.

How it's built

RunnAI is a standalone TypeScript app built on the Claude Agent SDK with a React/Ink terminal UI. It uses a 3-tier memory system: a hot cache loaded into every conversation, a deep memory store for detailed observations, and a local SQLite database with all your activity data. The agent decides what to remember, promote, or archive - inspired by MemGPT's self-editing memory pattern.
